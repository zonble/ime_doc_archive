# 附錄 B、自然語意分析實例

「自然輸入法」具有人工智慧的能力，他會分析您輸入文句的前後文，然後再從候選名單中，幫您挑選合適的字，省卻您字字修改的功夫。茲列舉四種範例，讓您熟悉語意分析是什麼。（記得：當您整句輸入完後，再修正文字，別讓「自然輸入法」沒表現的空間）

## 一、分辨同音詞

本<u>中心</u>訂於明日開張<br>
<u>衷心</u>感謝您的參與

## 二、分辨同音字

- 例一、「盡」、「進」同音兩字。<br>
  盡自己的力量<br>
  <u>進</u>自己的房間

- 例二、「向」、「像」兩同音字。<br>
  向台北警方自首<br>
  <u>像</u>台北警方同樣的能力

- 例三、「是」與「市」在相似句中。<br>
  台北<u>是</u>一個美麗的城市<br>
  台北**<u>市</u>是**一個美麗的城市<br>
  台北<u>市</u>有一個美麗的女孩

## 三、分析語句，斷詞功能

- 例一、三<u>歲到</u>五歲，<u>五到</u>十人
  <br>**隧道** 舞蹈
- 例二、台灣的<u>人工</u>很貴<br>
  他派兩**人供**我使用
- 例三、<u>忠孝</u>東路<u>五段</u>六十<u>一巷</u>五弄十七號九樓<br>
  **中校** **武斷** **一向** **酒樓**

## 四、中文數字辨認

- **十**月**十**日下午五**時**五分
- **十**月**十**日下午五**時十**五分
