# 漢音輸入視窗

- 狀態接紐：可顯示目前輸入模式•其模式顯示的代表符號如下所列：
  - 漢音模式：中
  - 地址模式：地
  - 英款全形：A
  - 英款半形：a
- 輸入視窗：可顯示注音符號（或拼音字母）及中文字•
- 游標：顯示輸入讀音或文字的位置•

由於本版漢音輸入法提供「行內游標眼随」與「浮動組字視窗，等翰入風格，所以在畫面上漢音視窗的樣式也
含有所差距・

- 行內游檩跟隨: 當選擇「行內游標眼隙」時，輸入視窗會在欲輸入的位置下方，而狀態接紐另外顯示。
- 浮新組字視窗: 當選擇「浮動組字視窗」時，可以調整輸入視窗大小，以利使用者輸入文字串。
