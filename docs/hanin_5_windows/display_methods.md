## 選擇讀音的輸入、顯示方式

輸入及顯示方式可利用下列方法切換及確定。

- 【鍵盤】：使用〔Tab〕鍵、〔↑〕鍵、〔↓〕鍵移動及選擇， 按〔Enter〕鍵結束。
- 【滑鼠】：在需要的選項上按一下選擇該項，按【確定】按鈕結束。

選擇讀音輸入及顯示方式的方法如下：

1. 進入漢音工具箱，打開電腦圖像。
2. 出現「漢音屬性設定」對話框。
3. 於「顯示方式」的選項按鈕中選擇所要的顯示方式。
4. 於「輸入方式」的選項按鈕中選擇所要的輸入方式。

漢音提供五種讀音輸入方式及兩種讀音顯示方式，您可以分別依自己習慣或喜好方式選擇。五種讀音輸入方式包括國語注音（標準鍵盤、倚天鍵盤及ＩＢＭ鍵盤）、國語注音第二式以及羅馬拼音。但如果您以注音為輸入方式，就無法以英文字母為顯示方式。除此漢音為了方便更多的使用者，另外提供了「[自定注音鍵盤](user_layout.md)」的功能。
