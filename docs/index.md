# 輸入法文件存檔

這份網頁存放了一些在 90 年代時的中文輸入法的說明文件，目的在於方便考古，確認當時各家中文輸入法的發展過程。

## 新注音輸入法 98

新注音輸入法是中文版 Windows 98 的系統輸入法，這裡取自 `C:\WINDOWS\SYSTEM\IME\TINTLGNT` 下的說明文件。原本是 CHM 格式，轉換成 Markdown 格式後放在這裡。

## 微軟新倉頡輸入法 98

新倉頡輸入法也是 Windows 98 的系統輸入法，這裡取自 `C:\WINDOWS\SYSTEM\IME\CINTLGNT` 下的說明文件。原本是 CHM 格式，轉換成 Markdown 格式後放在這裡。

## 漢音輸入法 5

在 Windows 安裝後，可以看到 `C:\HANIN` 目錄下的 Hanin95.help 檔案。
