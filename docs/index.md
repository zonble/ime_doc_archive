# 輸入法文件存檔

這份網頁存放了一些在 90 年代時的中文輸入法的說明文件，目的在於方便考古，確認當時各家中文輸入法的發展過程。

## 新注音輸入法 98

-   新注音輸入法是中文版 Windows 98 的系統輸入法
-   文件取自 `C:\WINDOWS\SYSTEM\IME\TINTLGNT` 下的說明文件。原本是 CHM 格式，轉換成 Markdown 格式後放在這裡。

## 微軟新倉頡輸入法 98

-   新倉頡輸入法也是 Windows 98 的系統輸入法
-   文件取自 `C:\WINDOWS\SYSTEM\IME\CINTLGNT` 下的說明文件。原本是 CHM 格式，轉換成 Markdown 格式後放在這裡。

## 倚天中文 2000 視窗

-   在安裝倚天中文 2000 後，放在 `C:\Etbox95` 目錄下的文件
-   軟體來自 [Internet Archive](https://archive.org/details/eten2000-for-cwin-95)

## 自然輸入法 99 （v 5.04）

-   在安裝自然輸入法 99 後，安裝目錄下有 HTML 格式的相關說明。
-   軟體來自 [Internet Archive](https://archive.org/details/going99)

## 漢音輸入法 5 macOS 版本

-   文件取自 Mac OS 9 系統資料夾中的檔案。
-   Mac OS 上的漢音輸入法缺少文件說明。

## 漢音輸入法 5 Windows 版本

-   在 Windows 安裝後，可以看到 `C:\HANIN` 目錄下的 Hanin95.help 檔案。
-   軟體取自 [Internet Archive](https://archive.org/details/HANIN502)。
-   在 Internet Archive 上，也可以找到[漢音 4 的安裝程式](https://archive.org/details/HANIN_402)，不過漢音 4 本身缺少文件說明。
