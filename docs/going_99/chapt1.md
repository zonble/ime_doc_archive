# 第一章、簡易入門

身為新手的您，可能迫不及待想使用「自然輸入法」V5.04，為使您在短短的 5 分鐘內領悟到「自然輸入法」帶給您意想不到的方便，請您跟著本章的步驟試一遍。

## 1-1、啟動輸入法

「自然輸入法」V5.04 與一般的輸入法啟動方式一樣，您可以滑鼠點選右下方的切換至，即可啟動「自然輸入法」V5.04，此時您在使用畫面會看到控制列（Function Bar）及輸入行（Input Line）兩個視窗。

## 1-2、使用實例

### 一、分析整句語意

我們以『注音輸入法』為例，請您試著輸入「我有一隻小花貓」整句，您是否注意到「隻」變化了幾次？

再試試輸入另一個句子「十月十日五時十五分」，有點繞口令，但是「自然輸入法」V5.04 挑選正確的詞給您。本產品具有整句語意分析功能，他能根據您輸入的前後文，自動調整選取的字，您不需要一個字一個字的選，當您輸入一句詞彙後，再檢查看看，而且您使用的次數越多，「自然輸入法」V5.04 越能瞭解您的心意。

### 二、更換同音字

接下來我們看看當「自然輸入法」V5.04 無法正確挑選您要的字時，該怎麼辦？請輸入「床前明月光」，由於「自然輸入法」V5.04 未曾學習過唐詩，將「明」誤認為「名」字，該怎麼辦？將游標移到「名」字上，試試鍵盤上的方向鍵「↓」效果如何？您可選「2」修正為「明」字。

### 三、尋找近音字

有時我們注音的發音不準確，捲不捲舌分不清時，怎麼辦？例如：「中華民國」錯打為「中華ㄇㄧㄥ ˊ 國」又用「↓」找不到「民」字怎麼辦？試試「↑」看看。「自然輸入法」為您準備了近音字，您不需要再為了「ㄔ」「ㄘ」「ㄥ」「ㄣ」傷腦筋。

### 四、挑選符號

您是否覺得使用文書軟體，輸入標點符號讓您很痛苦？試試 [Ctrl]+[7]，「自然輸入法」V5.04 為您準備了不少常用的符號，讓您更快速的輸入符號。使用「自然輸入法」V5.04，製作一份快速又完美的文件再也不是件難事。
