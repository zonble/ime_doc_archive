# 自建辭維護

使用者所登錄的自建辭登錄在自建辭庫內，利用自建辭維護工具可以管理維護自建辭。

■ 瀏覽
左側視窗列出目前的自建辭，您可用滑鼠按捲軸或用鍵盤（〔PageUp〕。［PageDown〕。〔↑〕。〔↓〕）來瀏覽所有的自建辭。

■ 尋找

1. 從右側視窗選擇一個注音符號。
2. 按【尋找】按鈕。<br>
   用滑鼠在注音符號上技一下可選擇注音符號，自建辭區是依照注音符號排序，右側視窗的注音符號是索引，系統會找出第一個讓音與該注音符號最接近的自建辭，診辭會反白表示被選擇到。

■ 刪除

1. 選擇該自建辭使它反白。
2. 按【刪除】按鈕。<br>
   用滑鼠在欲刪除的自建辭上按一下可選擇辭。<br>
   按【不刪除】按鈕可將刪除的最後一辭還原
