# 螢幕觸式鍵盤

螢幕觸式鍵盤使您可以用滑鼠來取代鍵盤，只要操作滑鼠，不須碰鍵盤就可輸入中文。

## ■ 啟動

啟動螢幕觸式鍵盤有下列２種方法。

- 由快速功能鈕進入<br>
  移動滑鼠指標到「快速功能鈕」中的「鍵盤」鈕上，點按滑鼠左鍵一下。
- 按[Ctrl+5]<br>
  當然您也可以更改此快速鍵的定義。

螢幕鍵盤的按鍵配置會自動更換，例如您更改讀音的輸入方式或改變輸入模式時，下列按鍵無法用滑鼠按下：

- 〔Caps Lock〕鍵
- 〔Num Lock〕鍵
- 左、右〔Shift〕鍵
- 左、右〔Ctrl〕鍵

## ■ 關閉

關閉螢幕鍵盤有下列方式：

- 用滑鼠左鍵點按兩下螢幕鍵盤（必須沒有顯示按鍵的部分）。
- 按〔Ctrl ＋ Space〕鍵可同時將螢幕鍵盤及漢音視窗關閉。
- 再用滑鼠左鍵點按一下快速功能鈕中的「鍵盤」鈕。

## ■ 大鍵盤

用滑鼠按螢幕鍵盤右上角的展開鍵「>>」。

螢幕觸式鍵盤可展開為大鍵盤，比原來的小鍵盤增加了九宮數字等鍵。

〔Insert〕、〔Delete〕、〔Home〕、〔End〕、〔PageUp〕、〔PageDown〕六個按鍵未加標註，但仍可使用。

## ■ 小鍵盤

用滑鼠按螢幕鍵盤右上角的縮小鍵「<<」。

按大鍵盤右上角的縮小鍵「<<」可讓螢幕鍵盤還原為小鍵盤。

## ■ 上移

以滑鼠按螢幕鍵盤上任何位置，或再啟動一次螢幕鍵盤，漢音視窗會顯示於所有視窗上面，不會被遮蓋（但可關閉），但螢幕觸式鍵盤可能被其他視窗遮蓋，這時可依上述操作將鍵盤上移。

## ■ 移動

以滑鼠拖曳螢幕鍵盤上非按鍵的區域。
