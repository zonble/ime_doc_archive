# 移除輸入法

若要移除微軟新注音輸入法，請按一下「開始」按鈕，並依序選擇「設定」，「控制台」。然後依下列方式移除：

1. 連按兩下 \[輸入法\] 圖示以開啟 \[鍵盤-內容\] 對話方塊。
2. 在輸入法清單方塊中選擇微軟新注音輸入法，按一下 \[移除\]，再按 \[確定\] 以確認移除。

使用此種方法只是將輸入法從使用者介面移除，該輸入法仍儲存在電腦硬體中，您可以再選擇
\[新增\] 將輸入法加入使用者介面。

**提示**

若您使用的是 Windows NT 5.0 作業系統，請按一下「開始」按鈕，並依序選擇「設定」，「控制台」。然後依下列方式移除微軟新注音輸入法：

1. 連按兩下 \[鍵盤\] 圖示以開啟 \[鍵盤-內容\] 對話方塊。
2. 選擇 \[輸入法\] 索引標籤，並在輸入法清單方塊中選擇微軟新注音輸入法，按一下\[移除\]，再按 \[確定\] 以確認移除。
