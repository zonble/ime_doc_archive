# 快速功能鈕

爲了方便使用者快速進入漢音所附屬的各項功能，我們提供了七種快速功能鈕，使用者只須在快速功能鈕的代表圖案上，點按滑鼠左鍵一下，便可以馬上進入這些功能中，節省您切換所需要的時間，並且可自由收放這七種快速鈕或選擇橫式排列、直式排列，不影響應用軟體的操作，除此也可以根據使用者的需求顯示個數，不佔畫面空間。

- 漢音：可隨時打開關閉另六種快速功能鈕，使用者可根據需要，自行打開或關閉其他六種快速功能鈕，
- 文字查詢：可快速進入文字查詢模態（請參考[文字查詢](query.md)）。
- 特殊符號選擇：提供使用者一些特殊的符號或圖案（請參考[特殊符號選擇](symbols.md)）,
- 重複句庫：顯示最近 8 次所輸入的不同句子（請參考[重複句庫](repeated_phrases.md)），
- 鍵盤：顯示目前鍵盤的排列方式，使用者可用滑鼠來取代鍵盤，只要操作滑鼠，不須碰鍵盤就可輸入中文（請參考[螢幕觸式鍵盤](screen_keyboard.md)），
- 漢音工具箱：顯示漢音工具箱（請參考[漢音工具箱](toolbox.md)）
- 輔助說明視窗：提供漢音輸入法線上輔助說明，

對於快速功能鈕您也可以設定出現的個數，設定步驟如下：

1. 點按「漢音工具箱」快速鈕，出現「漢音輸入法」對話框。
2. 點選「快速功能鈕設定」按鈕，
3. 選擇想要展開的快速功能鈕，並可選擇是否要使用較大按鈕及橫式排列等，
