# 重複字庫

漢音提供重複句庫的功能，它會自動記憶最近輸入的 8 個不同句子，當使用者要重複輸入前 8 次中所輸入過的句子時，只要利用滑鼠直接點擊重複句庫快速功能鈕或按［Ctrl+4］，便可以從顯示中的句庫裡選出所要的句子，無須再執行鍵盤輸入。

例如至目前為止前 8 次的輸入中己經輸入過〝產品企劃部”，現在要再次輸入時，可以利用重複字庫。其方法如下：

1. 點擊重複句庫快速功能鈕（或按［Ctrl+4］），重複句庫視窗內會顯示最近記憶的 8 個不同句子。
2. 從重複句庫視窗內選擇所要的句子〝產品企劃部」，按［Enter］送回輸入視窗。
