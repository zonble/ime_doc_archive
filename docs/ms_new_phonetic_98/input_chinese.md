# 輸入中文

您只需直接輸入注音，微軟新注音輸入法就會根據您輸入的字及其前後文的關係自動調整正確的字詞。輸入法可以自動調整的前後最長距離是三十一個字，超過三十一個字後，輸入法會自動將最前面的長詞送出，若於長詞後面緊接著的是無法開啟同音字詞候選視窗的字元，如半形符號，包括數字、字母及符號﹔還有全形的數字及英文字母，則會將這些字元一併送出。

您可以使用 Enter 鍵或 [字尾符號](endsymbol.md)
來完成輸入，而在完成輸入之前若有任何字詞需要修改，您可以移動游標至該字並使用方向鍵來進行編輯的工作。有關如何修改輸入的文字，請參閱
[挑選同音字詞](pickup.md) 。

**提示**  
如果您需要重新輸入同一個字，可以在句子未完成輸入前按下 Ctrl + Alt + K
鍵。如果您需要重複輸入剛剛完成的句子，請按 Ctrl + Alt + G 鍵。
