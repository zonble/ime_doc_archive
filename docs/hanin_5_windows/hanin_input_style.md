# 输入風格設定

## 設定智慈型提示

漢音WIN95版提供2種智慧型提示。

- 辭彙變換: 當變換結果為辭彙時，敌辭彙下會顯示紅色波紋底線。
- 略語選擇: 執行略語辭鍵翰入時，若為正確的已登綠略語辭鍵，則鼓字串下會顯示藍色波紋底線。設定的方法如下：
    1. 進入漢音工具箱
    2. 打開電腦國像。出現「漢音屬性設定」對話框。
    3. 點送「輸入風格」核對框中的智慧型提示。當漢音2字下顯示紅色波紋底線時，表示設定完成．

##  選擇輸入風格

漢音WIN9S版提供2種輸入風格•
- 行內游標眼隨: 所謂的「行內游標眼」（即所謂的in-line輸入）是指跟隨使用中應用軟內的翰入女字游標，直按翰入文字於應用軟體內⋅
- 浮動組字視窗: 文字不直按輸入於用軟中，而是透過輸入的愛換視窗後，再將女字串送至使用中的應用軟艦上•

選擇翰入風格的方法如下：

1. 進入漢音工具箱。
2. 打開電腦固像。出現「漢音屬性設定」對話框。
3. 於「輸入風格」的選項接紐中選擇所要的翰入方法．
