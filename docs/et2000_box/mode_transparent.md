# 透通模式

透通模式互換鍵(Transparent Mode Toggle)

《使用方式》

請選一種操作方式

-   ▲ 選「設定」下拉表直接選透通模式，在前方會出現「ˇ」。
-   ▲ 用滑鼠選 ![bm2.BMP](./bm2.png) 按鈕。
-   ▲ 用鍵盤按[Alt]-[Ctrl]-[T]鍵。

※不使用透通模式

-   英文 DOS 下，游標在字間移動時，碰到英文字一次僅移動１個位元組，但碰到中文時，則一次移動２個位元組。
-   中文 DOS 下，碰到英文字一次僅移動１個位元組，但是碰到中文時，則一次移動４個位元組。

※使用透通模式

-   英文 DOS 下，無論是碰到中英文字，游標都只移動１個位元組。
-   中文 DOS 下，碰到英文字一次僅移動１個位元組，但是碰到中文時，則一次移動２個位元組。
