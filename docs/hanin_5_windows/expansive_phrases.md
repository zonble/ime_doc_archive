# 略語辭

對於一些經常使用的辭句，例如地址或是公司名稱等，使用者可建立為略語， 以減少輸入的時間。

只要將辭句指定一個您容易記得的中文（二個或三個中文字）或英文（二個或三個全形英文字母）略語鍵登錄，當需要這些辭句時，輸入略語鍵後再執行[略語展開](expand_phrases.md)功能（按[Tab]鍵），就可以展開原先登錄的長辭或短辭。之後再按[Enter]鍵送出字串至輸入視窗。關於略語的詳細操作請參考略語展開功能及[略語辭維護](expansive_phrases_maintain.md)。
