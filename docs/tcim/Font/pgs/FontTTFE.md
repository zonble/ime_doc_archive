# 字元編碼

中文大五碼分成三部份，實際使用範圍是：

|       |                                                                     |
| ----- | ------------------------------------------------------------------- |
| **1** | 符號：$A140 至 $A3FE                                                |
| **2** | 常用字：$A440 至 $C67E 共 5,401 個編碼<br>保留區：$C6A1 至 $C8FE    |
| **3** | 次常用字：$C940 至 $F9D5 共 7,652 個編碼<br>保留區： $F9D6 至 $FCFE |

造字區為：

|       |                          |
| ----- | ------------------------ |
| **1** | 符號：$A3C0 至 $A3FE     |
| **2** | 常用字：$C6A1 至 $C8FE   |
| **3** | 次常用字：$F9D6 至 $FCFE |

使用者造字時了解大五碼分佈是很重要的，非造字區有標準的大五碼字元，而造字區一般是空白的，可供使用者儲存新造的字。如果使用者把新造字放在非造字區，在輸入時便很容易引起混亂。

“TrueType 字體編輯程式”在使用者造字時，預置使用造字區的編碼（稱為“正規字”)，隨著造字的增加，會在造字區順序編排下去。但使用者可以更改造字的編碼，這時便要小心分別造字和非造字區。

一般情形下，我們把大五碼標準的 13,053 字元以外的字元，稱為“外字”，使用者可以自行定義。
