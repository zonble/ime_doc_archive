# 自訂鍵盤符號對應

鍵盤符號對應亦為微軟新注音輸入法的新增功能之一，您可以在
[輸入法狀態視窗](inputwindow.md) 按一下滑鼠右鍵，並在功能表中選擇
\[內容\] (Properties)，進入 \[鍵盤符號對應\]
索引標籤。然後根據對話方塊內的設定，將系統預設的符號對應依個人的需要變更為自訂的鍵盤符號。本輸入法提供了全半形標點符號、特殊符號、數學符號、單位符號、希臘字母以及注音符號等七種供您選擇。請依下列步驟自訂鍵盤符號：

1. 先設定要變更符號對應的環境：在鍵盤上選擇 CapsLock 鍵 On 或 Off 的狀態 (On 為鍵
   盤上層的符號；Off 為鍵盤下層的符號)，並在輸入狀態視窗按一下全半形的選項(預設為
   半形)。(注意：您所作的符號變更只會在您此時選定的環境設定下啟動)
2. 如果您需要一次同時變更鍵盤上下層及全半形的符號，可以選擇 \[預先定義的符號對應
   \] (Predefined symbol mapping) 方塊中的選項，該選項是選擇性的，亦即可選可不
   選。此功能旨在省去您逐個變更的繁複工作。例如，您希望在 CapsLock 鍵 Off 及半形
   的環境下所輸入的全都是鍵盤上層且是全形的符號，只要選擇\[鍵盤上層之全形符號\]
   (Full shape symbol at keyboard top) 選項就可以了，無須逐個變更。以後當您進入
   CapsLock 鍵 Off 及半形的輸入狀態時，按下 Shift 鍵，再按符號，都會出現全形的鍵
   盤上層符號，而非預設的半形鍵盤下層符號。
3. 在鍵盤上按一下您要變更的符號或數字 (顯示為黑色)。
4. 在 \[符號群\] (Symbol group) 下拉式方塊中選擇符號種類。
5. 在 \[可選用符號清單\] (Symbol list)
   方塊中連按兩下您要替代的符號。再按一下 \[確定\]。
6. 如果要恢復系統預設值，請按一下 \[重設\] (Reset) 按鈕。
