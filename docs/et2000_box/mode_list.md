# 條列模式

條列模式互換鍵(List Mode Toggle)

《使用方式》

請選一種操作方式

-   ▲ 選「設定」下拉表直接選條列模式，在前方會出現「ˇ」。
-   ▲ 用滑鼠選 ![bm4.BMP](./bm4.png) 按鈕。
-   ▲ 用鍵盤按[Alt]-[Ctrl]-[L]鍵。

本系統提供給使用者透過某些特殊的控制碼處理印表或繪圖方面的功能，因此在一般正常狀況下，只要下達這些印表或顯示方面的控制指令，系統都會立即加以解釋，並反應出結果。

但如果您只想了解各指令設定的內容，而不希望獲得實際執行的結果時，則可切換到對系統控制字元不反應的條列模式下，那麼系統便會將這些控制指令視為一般的文字敘述而條列出來。

在條列模式下，系統不會去解釋印表控制字元「～」所帶領的指令，只會將原本如「～ U2 ；」的控制指令視為一般字元看待。
