# 選擇同音字辭

在漢音輸入法 5.0 版執行中輸入時，如果您在漢音工具箱的輸入風格中是選擇「智慧型提示」，則目前變換結果為辭彙時，該辭彙下會顯示紅色波紋底線，提醒使用者可用［Space］選擇是否有同音字辭或可按［Esc］ 來取消轉換。（請參考[輸入風格設定](hanin_input_style.md)）

## 使用鍵盤選同音字

1. 移動游標到字（辭）上。
2. 按空白或〔↓〕顯示同音字（辭）。
   - 畫面會出現「同音字辭選擇」的視窗，視窗內顯示著所有的同音字辭。倘若同音字辭過多，無法全部放入一個視窗內時，視窗左下角會顯示所有同音字辭的視窗數目，及目前在第幾個視窗下。
   - 使用者可利用〔Page Up〕鍵及〔Page Down〕來切換顯示視窗。（也可以利用〔↑〕、〔↓〕、〔Space〕鍵慢慢地切換顯示視窗，直到您需要的字辭出現）
3. 按該同音字（辭）前對應的英文代表鍵（A..E）及數字鍵（1..6），或用〔↑〕、〔↓〕、[←]、[→]來選擇所要的字（辭），除此還可利用滑鼠在該同音字（辭）上點一下。

當輸入視窗內的字（辭）轉換不合要求時，您仍可選擇同音字（辭）。

**當游標位於字串之後時，直接按下空白鍵（不須移動游標）就可以顯示最後一辭的同音辭（如果以最後一字結尾的辭在辭庫中有同音辭）或最後一字的同音字，如果同音辭與同音字都存在，則先顯示辭後顯示字。**

當游標在字上時，則顯示游標上字（辭）的同音字（辭），如果游標後還有字，且以游標上字為首的辭有同音辭，則系統先顯示同音辭，後顯示游標上字的同音字。當顯示一字（辭）的同音字（辭）時，該字（辭）會反白。

如果您想要的字沒有顯示，要看其他的同音字（辭），可以用〔+〕鍵、〔-〕鍵、［Space］鍵，或用［Page Up］、［Page Down］來切換顯示視窗。視窗一次最多出現三十個同音字，您可在所有同音字間上下移動。

當視窗顯示同音字（辭）時，您可以按【Esc】鍵，即可由選擇狀態回到輸入狀態。

## 使用滑鼠選同音字

現在，漢音提供了滑鼠選字功能，只要在輸入視窗內的任一字上左鍵點擊兩下，即會列出同音字供選擇，相當於將游標移到此字上，並按下空白鍵的效果。同樣的，在任一同音字上點擊一下左鍵，即選取了此同音字，點擊兩下則取代原字，相當於按下同音字前的選字鍵。
