# 漢音工具箱

使用者可由漢音工具箱選擇讀音的輸入、顯示方式﹑輸入風格設定、自用辭庫的管理、自建辭及略語辭之維護﹑設定自定注音鍵盤﹑專業辭庫等細節。

- 由快速功能鈕進入<br>
  移動滑鼠指標到「快速功能鈕」中的「漢音工具箱」鈕上，點按滑鼠左鍵一下。
- 由控制台的輸入法管理員中進入<br>
  由控制台的輸入法管理員中選擇漢音輸入法的 5.0 版，按[內容]按鈕開啟漢音工具箱視窗。<br>
  出現「漢音輸入法」對話框。

不論由何者進入工具箱畫面，當出現「漢音輸入法」對話框後點按【工具箱】按鈕。

漢音原則上預設為工具箱視窗，如果不在工具箱視窗時，請點按左上角的【工具箱】按鈕。

漢音工具箱視窗出現後，只要將滑鼠指標移到各個圖像功能上，便會顯現此圖像功能的簡單說明，點按圖像一下就可執行該圖像的功能。
