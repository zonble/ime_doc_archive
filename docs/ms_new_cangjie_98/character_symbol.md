## 標點符號

由於倉頡輸入法所使用的鍵盤設定，並未與英文鍵盤衝突，所以在輸入全形或半形標點符號時，只需根據全半形的狀態直接按鍵盤上所對應的標點符號即可。下表列出常用的標點符號對應：

|                 |            |             |
| --------------- | ---------- | ----------- |
| 標點符號        | 按鍵       | 全/半形狀態 |
| 全形逗號 (，)   | ','        | 全形        |
| 全形句號 (。)   | '.'        | 全形        |
| 全形分號 (；)   | ';'        | 全形        |
| 全形問號 (？)   | Shift +'/' | 全形        |
| 全形冒號 (：)   | Shift +';' | 全形        |
| 全形驚嘆號 (！) | Shift +'1' | 全形        |
| 半形逗號 (,)    | ','        | 半形        |
| 半形句號 (.)    | '.'        | 半形        |
| 半形分號 (;)    | ';'        | 半形        |
| 半形問號 (?)    | Shift +'/' | 半形        |
| 半形冒號 (:)    | Shift +';' | 半形        |
| 半形驚嘆號 (!)  | Shift +'1' | 半形        |
