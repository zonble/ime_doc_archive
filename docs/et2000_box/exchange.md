# 互換鍵

|                                     |                                                                                      |
| ----------------------------------- | ------------------------------------------------------------------------------------ |
| [透通模式](mode_transparent.md)     | 透通模式互換鍵(Transparent Mode Toggle) <br>可利用鍵盤按[Alt]-[Ctrl]-[T]鍵切換。     |
| [內碼輸出](mode_hex.md)             | 十六進位模式互換鍵(Hexadecimal Mode Toggle) <br>可利用鍵盤按[Alt]-[Ctrl]-[H]鍵切換。 |
| [條列模式](mode_list.md)            | 條列模式互換鍵(List Mode Toggle) <br>可利用鍵盤按[Alt]-[Ctrl]-[L]鍵切換。            |
| [自動判斷表格](mode_auto_table.md)  | 自動判斷英文表格符號功能互換鍵 <br >可利用鍵盤按[Alt]-[Ctrl]-[Z]鍵切換。             |
| [純英文模式](mode_plain_english.md) | 中文模式與英文模式互換鍵 <br>可利用鍵盤按[Alt]-[Ctrl]-[A]鍵切換。                    |
