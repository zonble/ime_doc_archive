# 自建辭

漢音提供線上登錄自建辭的功能，使用者可以隨時追加新辭。使用者所登錄的自建辭登錄在[自用辭庫](user_phrases.md)內，利用[漢音工具箱](toolbox.md)可以管理維護或更換不同的自用辭庫。

漢音辭業翰入法雖然有很高的轉換正確率，但是對於非連續性辭句，如人名。街地名等專有名辭，法律，醫學等專業術，俚語、綃語、簡稱。俗語等因文化背景、時問變化所新增、消滅之辭彙如「頂客族」、「柏青哥」、「忍者龜」等，漢音輸入法無法盡搜歸納。由於加不勝加，減不勝減，漢音輸入法便提供這種簡便線上登錄自建辭，您可以很容易的定義您需要的新辭。

## 自建辭登錄

當游標位於中文字串倒數 2~5 個字上，按〔Shift + Enter〕艇。自游標位置算起至字串最後一字止的辭就會登錄於自用辭庫中的自建辭區，登錄完後，游標會自移到最後。

如果自用辭庫的自建辭區已經有您想要登錄的辭時，系統啤嗶叫，但游標仍會自移到最後。如果您按［ Shift+ Enter〕鍵後系統嗶嗶叫，游標不會自動移到最後，就是表示登綠失敗，可能有以下原因：

- 如果要登錄的自建辭太短或太長時（長度應為 2-5 字）。
- 自用辭庫的自建辭區已經滿了。
- 漢音工具箱已經將自建辭維護工具打開，還沒有關閉以前，是無法登錄自建辭。
- 您想要登錄的辭有英數符號或標點符號。

漢音工具箱的自建辭維護工具，提供對自建辭的尋找、刪除、瀏覽三種功能，請參考漢音工具箱中的『自建辭維護』。

例：試將「忍者陷」登錄為自建辭。

1. 移動游標到「忍」字
2. 按〔Shift +Enter〕鍵登錄自建辭「忍者龜」

如果您第使用「忍者龜，一辭的話，將它登條在自用辭庫的自建辭區內，下次您再輸入讓音「ㄖㄣ ˇ ㄓㄜ ˇ ㄍㄨㄟ」時，就會直按轉換得到「忍者龜」•
