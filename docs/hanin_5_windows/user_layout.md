# 自定注音鍵盤

漢音在注音輸入方式時除基本提供三種內定鍵盤（標準鍵盤、倚天鍵盤及 IBM 鍵盤）外，更可依使用者自己的習慣或喜好設計出各式各樣不同的鍵盤組合。在漢音所提供的「使用者自定注音鍵盤」功能下，使用者可隨時依注音規則自訂自己的注音排列方式，並且為方便管理這些鍵盤，使用者可對每個自定鍵盤定義中文名稱。

## 進入自定鍵盤的方法

1. 進入漢音工具箱。
2. 打開電腦圖像。<br>出現「漢音屬性設定」對話框。
3. 於「輸入方式」的選項按鈕中選擇「自定注音鍵盤」。<br>出現「使用者自定注音鍵盤」的視窗。

---

注意：若要打開已定義好的自定鍵盤，則先選定好鍵盤名稱後再按「自定注音鍵盤…」。

---

## 離開自定鍵盤的方法

- 【鍵盤】：按［Esc］。
- 【滑鼠】：點按〔 結束〕鈕。

## 自定注音鍵盤的方法

使用者可依照鍵盤一注音規則自定自己的注音排列方式。

1. 進入漢音工具箱。
2. 打開電腦圖像。<br>出現「漢音屬性設定」對話框。
3. 於「輸入方式」的選項按鈕中選擇「自定注音鍵盤」。<br>出現「使用者自定注音鍵盤」的視窗。
4. 選擇『建立新鍵盤』。
5. 利用滑鼠由右下方的注音表中將所選到的注音符號拉到欲定義的按鍵位置上。
   - 已被定義的按鍵會變色，而右上角的區域（其上方的文字或數字表示原來按鍵的符號）會顯示目前按下的按鍵及其所定義的注音符號。由於本功能對每個鍵不限定只能定義 1 個符號，為了讓使用者能清楚了解目前已定義的符號，所以右上角的區域共有四個方格供顯示。
   - 除此若您所定義的規則不合理時，系統會發出「呷」聲後，被拖曳的注音符號會自定跳回原來的注音表中，表示此項定義不被接受。
6. 選擇「〔更名〕儲存新鍵盤..」。

- 執行步驟 6 後，畫面會出現「儲存新鍵盤」的對話框。使用者可對此自定鍵盤輸入鍵盤名稱。

---

注意：若沒完全定義完成，無法執行儲存。

---

## 自定注音鍵盤的規則

首先使用者必須將注音定義在 48 個合法按鍵上，至於要「一鍵一音」或「一鍵多音」，完全隨使用者高興，但 42 個注音都必須定義，否則某些注音便無法輸入，這當然是一組不合法的鍵盤。

檢查合不合法的方法很容易，只要檢查 48 個合法鍵：

1. 如果沒有注音或僅有一個注音定義在上面，則沒有問題。
2. 如果有兩個或兩個以上的注音定義在上面，那就要將這些注音兩兩做檢查，根據表格看看它們所屬的組群相不相容，有任何不相容就不是合法的注音鍵盤。
