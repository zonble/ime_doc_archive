# 漢音文字查詢

遇到不會發音的字時，可利用文字查詢來解決問題。

執行此「文字查詢」有下列二種方法：

1. 將滑鼠移到快速功能鈕中的「文字查詢」鈕後，點技滑鼠左鍵一下。
2. 接下［Ctrl+2〕。

啓動後，出現的『漢音-文字查詢』畫面。如同查字典的方法一樣，您可以選擇被查詢字的部首或部首＋筆畫，或輸入總筆畫，然後按〔尋找〕接鈕。

漢音會預設使用都首（在『使用部首』前的方格內畫有「V」，即表示「設定」的意恩），如果您不要選用都首，只要將滑鼠指標移到方格內後，點接滑泉左鍵一下，當方格內的「V」消失時，即表示此功能無效（或接［Ctrlt））。

按著畫面顯示所有符合條件的字。搀動搀轴或接下方向鍵，過擇您要詢間的文字。然後您可以執行以下功能：

1. 按〔詢問〕接鈕，顯示要詢問文字的注音，部首或筆畫等資料視窗。
2. 按〔送回〕按鈕，可將詢問文字送回輸入視窗。此時文字查詢也會自動結束，如果您並未執行送回指令，可繼續執行下列步聚。
3. 技〔尋找〕技鈕，回到『都首筆畫』畫面。
4. 按〔結束〕按鈕，結束文字查詢。

## 查詢注意事項

1. 文字查詢時，各功能所使用的快速辣（HOTKEY）須加按 CtrL 糙，例如欲『詢問』時按［C1rl +F〕才能正常動作。
2. 文字查詢時，當漢音在選擇、略、翰入注音或輸入視窗已填滿等狀態下，無法將文字送回漢音輸入視窗。
3. 如果越文字無讀音時，漢音無法修政以文字查詢方式所取得文字之注音。
