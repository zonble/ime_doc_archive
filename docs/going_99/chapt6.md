# 第六章、答客問

## 問題一、「自然輸入法」自動學習的輔助辭典，存放在哪裡？是否可自行備份保存？

答：

「輔助辭典」內的資料是存放在 `<安裝目錄>\USER.UDB` 中。

如果您想備份相關的學習檔，建議您同時備份下列各檔（如果存在）：

-   `USER.LRN`：學習暫存檔
-   `USER.DIC`：輔助辭典文字檔
-   `USER.BOX`：略語輸入文字檔
-   `*.UDD`：自建詞庫檔

## 問題二、如果將「自然輸入法」設定為開機時內定輸入法，會造成開機當機，該怎麼辦？

答：

請依下列步驟執行：

1. 開機時，先進入 MS-DOS 模式（不啟動 Windows），將 `C:\Windows\System` 下的輸入法檔案更換檔名。
2. 鍵入 `WIN`，進入 Windows 環境。
3. 進入「控制台」→「鍵盤」，更改開機時內定輸入法為「EN」。
4. 關機回到 MS-DOS 模式，將檔名還原。
5. 重新回到 Windows 即可。

各版輸入法檔案如下：

-   「自然輸入法」專業版：`GOING5.IME`
-   「自然新注音」：`GOPH5.IME`
-   「自然新拼音」：`GOPY5.IME`
-   「自然推廣版」：`GOHSU5.IME`

## 問題三、為何無法在「MS-DOS 模式」中，輸入中文字？

答：

「自然輸入法」使用環境為視窗環境，送出的中文字，在 Windows 95/NT 中文版中，並不直接傳給 MS-DOS 模式。

## 問題四、在 MS-Word 中使用「自然輸入法」，輸入小寫英文字，為何部分字母自動變為大寫？

答：

請在 MS-Word 的「工具」選項裡，將「自動校正」功能關閉，則輸入與顯示就會相同。

## 問題五、解安裝時，為何無法移除？

答：

在解安裝時，如果仍有其他應用軟體正在使用，解安裝程式便無法完全執行。（相關說明請參閱[第四章「勤前教育」](chapt4.md)）

## 問題六、如何關閉自動學習？

答：

請點選「控制列」裡的 ![](images/Icon-Go-Learn.gif)，當呈現灰階時，表示停止自動學習功能。

## 問題七、中英文切換是否有熱鍵（快速鍵）？

答：

`[CapsLock]` 就是您要的答案。使用快速鍵切換，比使用 Windows 95/NT 中文版關閉輸入法來得快速。相關說明可參考[熱鍵列表](Chap-Aa.md)。

## 問題八、為何解安裝，再安裝後，重新開啟「自然輸入法」，卻無法使用？

答：

在 Windows 95 的環境裡，系統程式是可共用的。在您解安裝時，可能其他應用程式中有使用「自然輸入法」，造成解安裝不完全。此時請您關閉所有應用程式，重新解安裝「自然輸入法」，再重新啟動系統，即可使用「自然輸入法」。（相關說明請參閱[第四章「勤前教育」](chapt4.md)）

## 問題九、如果遇到技術問題或其他使用上的問題時，該如何尋求解答？

答：

在您詢問技術問題前，請先準備下列資料：

-   硬體配備：PC 廠牌、CPU、RAM、聲霸卡、其他配備
-   使用中軟體：例如 `MS-WORD`
-   產品版本：請在「控制列」中的「關於」項內獲得版本編號
-   產品序號：包裝盒上的序號
-   聯絡方式或電話：如果需要回覆時
-   問題產生過程或現象：以利驗證、測試，找出問題
