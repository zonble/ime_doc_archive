# 標點符號

您可以使用下列方式輸入全形符號：

1. 在輸入符號之前，先輸入前導字元 (\`)。例如，若要輸入全形的句號 (。)，可以輸入 '\`' + '.'。
2. 使用 Ctrl+Alt+，(逗號) 啟動 [符號鍵盤](symbol_keyboard.md) (Symbol
   Keyboard) 的螢幕小鍵盤，並使用滑鼠按一下該鍵盤上的全形符號。

**提示**  
使用符號鍵盤，一次只能選擇一個符號，之後該鍵盤會自動關閉。若要再鍵入符號，必須重新啟動符號鍵盤。

根據全形或半形及 CapsLock 鍵 On 或 Off 不同的狀態下，符號與鍵盤具備不同的配置。下列為常用之標點符號的鍵盤配置：

| 標點符號        | 按鍵        | 全/半形狀態 | CapsLock 狀態 |
| --------------- | ----------- | ----------- | ------------- |
| 全形逗號 (，)   | Shift + ',' | 全形        | Off           |
| 全形句號 (。)   | Shift + '.' | 全形        | Off           |
| 全形分號 (；)   | Shift + ';' | 全形        | Off           |
| 全形問號 (？)   | Shift + '/' | 全形        | On            |
| 全形冒號 (：)   | Shift + ';' | 全形        | On            |
| 全形驚嘆號 (！) | Shift + '1' | 全形        | On            |
| 半形逗號 (,)    | Shift + ',' | 半形        | Off           |
| 半形句號 (.)    | Shift + '.' | 半形        | Off           |
| 半形分號 (;)    | Shift + ';' | 半形        | Off           |
| 半形問號 (?)    | Shift + '/' | 半形        | On            |
| 半形冒號 (:)    | Shift + ';' | 半形        | On            |
| 半形驚嘆號 (!)  | Shift + '1' | 半形        | On            |
