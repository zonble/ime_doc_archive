# 重複字庫

漢音提供重句庫的功能，它會自動記億最近輸入的 8 個不同句子，當使用者要重複翰入前 8 次中所輸入過的句子時，只要利用滑鼠直按點接重複句庫快速功能鈕或接［Crl+〕，便可以從顯示中的句庫裡選出所要的句子，無須再執行鍵盤翰入。

例如至目前為止前 8 次的翰入中己經翰入過〝產品企劃部”，現在要再次輸入時，可以利用重複字庫。其方法如下：

1. 點技重句审快速功能鈕（或接［Ctl+4］）重句庫視窗內會顯示最近記億的 8 個不同句子。
2. 從重複句庫視窗內選擇所要的句子〝產品企些部。按［Enter］辣送回翰入視窗。
