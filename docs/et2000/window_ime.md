# 倚天輸入法

進入倚天視窗輸入法設定程式中，提供了未安裝輸入法、已安裝輸入法的選項和設定的按鈕。

在安裝好的輸入法，可利用『設定』的按鈕，選擇輸入法提供的功能。

|                                    |                                            |
| ---------------------------------- | ------------------------------------------ |
| [倚天倉頡輸入法](canjie.md)        | 提供[設定使用狀態](canjie.md)。            |
| [倚天注音輸入法](window_layout.md) | 提供選擇[注音鍵盤](window_layout.md)種類。 |
| [倚天忘形輸入法](etai.md)          | 提供選擇[多種設定模式](etai.md)。          |
| 倚天簡易輸入法                     | 不提供設定功能。                           |
| 倚天羅馬輸入法                     | 不提供設定功能。                           |

如果您尚未安裝倚天輸入法的話，所有的輸入法會顯示在『未安裝輸入法』顯示區中，利用滑鼠直接選要使用的輸入法後，再用滑鼠點選 → 按鈕，將選取的輸入法移到『已安裝輸入法』顯示區中，即可用切換方式使用安裝好的輸入法。

如不需使用的輸入法，可直接選『已安裝輸入法』顯示區中的輸入法後，再用滑鼠點選 ← 按鈕，將選取的輸入法移到『未安裝輸入法』顯示區中，即可刪除輸入法。

※ 切換輸入法時，請離開倚天輸入法設定程式。
