# 5.0 版特性概說

1. 為完全 32 位元程式，每一個可輸入視窗單位(thread)可擁有一個漢音輸入法，可隨時切換使用。
2. 行內游標跟[^inline]隨輸入風格，目前可編輯領域（未確定組字文字列）由於顯示速度等關係，暫定為 10 個字。您最好把文字輸入字形調成與您應用程式一樣比較好。（漢音可隨時改變輸入字形）。初期，您也許不習慣行內輸入風格，如果您習慣行內游標跟隨輸入(InLine)輸入以後，您會覺得以後輸入中文會很順暢，不蓋您的！
3. 新功能[“Shift”](input_mode.md)鍵為中英文切換使用，如果您在漢音內只按“Shift”又放開（不含任何其他鍵）漢音便會自動切為半形英數，您可自由輸入英數，又按“Shift”又可變回漢音輸入狀態。有些軟體可自動辨別英數或中文，但如果您輸入碰巧按錯鍵，它便硬變成英數，實在不是很實用，因為輸入者本身最清楚是否要變成英數，是否輸入英數應由使用者決定才是。
4. 漢音設計原則是，輸入法應以最少的“篇幅”來輸入，不應佔很大的位置，擋住您應用軟體的操作，儘量讓使用者的應用軟體得到最大操作範圍的舒服感，所以漢音所設計的快速功能鈕可以收放移動，可以按照使用者熟悉漢音功能的多寡來選擇功能鈕的多寡，愈使用漢音愈可了解漢音功能性的強大，讓一般的追隨者望塵莫及。無窮的應用樂趣，留待使用者去發掘。
5. 5.0 版保持 4.02 版功能，新增的功能有
   - [輸入風格](hanin_input_style.md)的選用。
   - 單鍵“Shift”切換英數／漢音
   - [重複句庫](repeated_phrases.md)的選用
   - [自定功能鍵](keys_customize.md)，[功能鈕](buttons.md)
   - 線上使用說明
   - 追加旅遊專業辭典，可輸入觀光旅遊名勝如“錫安公園”﹑“優勝美地”﹑“水濂洞”﹑“睡獅岩”等，以及世界各國首都名稱等。

注意：

1. 您不要任意去修改“Win95”內“Whanin.ini”內容，或刪除它，否則有些功能可能會失效。
2. 漢音程式名稱：

   ```text
     ■Hanin目錄下：
       Hanin95.ime      漢音輸入法程式（放在“Win95\system”）
       Query95.exe      漢音文字查詢程式
       Tools95.exe      漢音工具箱程式
       Whanin.ini       漢音屬性記錄檔（放在“Win95”）
       *.dic            辭典（主辭典﹑地址／曖昧／人名／部首筆畫辭典）
     ■Usr_file目錄下
       記錄使用自定鍵盤檔案$usr***.kbd
       記錄使用者自建詞及略語﹑學習檔案﹑$usr***.dic
     ■Expert目錄下
       專業辭典檔案*.dic
   ```

3. 如果您將漢音工具箱按鈕由快速按鈕欄中刪除後，您要再使用工具箱時，只要從控制台輸入法中漢音 5.0 的“內容”進入選用即可。

[^inline]: 行內輸入風格: 當選擇「行內游標眼隨」時，輸入視窗會在欲輸入的位置下方，而狀態接鈕另外顯示。請參考[輸入視窗](hanin_input_window.md)或[輸入風格設定](hanin_input_style.md)。
