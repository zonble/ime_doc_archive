# 輸入模式

漢音提供四種輸入模式，視窗的狀態按鈕會顯示您目前處於哪種模式。關於此四種模式切換的方法參考[切換輸入模式](input_mode_switch.md)。

## 漢音輸入模式

在此模式下，狀態按鈕顯示「中」。您所鍵入的碼代表中文的讀音，當您鍵完一個字的完整音後，系統會自動把讀音轉換成該讀音的最優先選字，或在必要時與前字做辭句轉換。

## 全形輸入模式

在此模式下，狀態按鈕顯示「A」字，您所鍵入的碼代表全形的英數。

## 英數輸入模式

在此模式下，狀態按鈕顯示「a」字，您所鍵入的碼代表一般的半形英數碼，會直接透給應用程式，不影響漢音視窗內的文字。

## 地址輸入模式

在此模式下，狀態按鈕顯示「地」字，漢音的辭彙自動轉換會參考「地址專用辭庫」。此時輸入地址的轉換正確率將大為提高。例如，在漢音輸入模式下，輸入「台北市士林區菁嚳路」，會變成「台北市士林區經律路」，但在地址輸入模式就沒有問題。
