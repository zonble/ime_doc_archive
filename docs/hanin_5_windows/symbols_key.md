# 特殊符號功能鍵

使用漢音輸入全形特殊符號，除了可由快速功能鈕中的「特殊符號」選擇所要的符號以及由鍵入的代表讀音去找尋相關的特殊符號外，漢音另提供了更完整的 “\” 功能鍵特殊符號對照表（請參考使用手冊的附錄）。欲輸入特殊符號，只需先按下〔\〕，再按下其他鍵，即可將所對應的特殊符號列出以供選擇。

例：列出 A 鍵上的特殊符號

1. 按下特殊符號功能〔\〕。<br>
   此時輸入視窗內會顯示“符”字。
2. 按〔A〕，並按下空白鍵，將列出 A 鍵的特殊符號。

每個鍵所對應的特殊符號，都包括了此鍵面上的符號，如倉頡字形、注音符號、英文字母等。在自定注音鍵盤輸入方式下，您可以用此項功能鍵輸入特殊符號。
