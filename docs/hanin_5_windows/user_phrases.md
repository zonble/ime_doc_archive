# 自用辭庫

使用者所登錄的自建辭、略語與字、辭的自學習資料均記錄在自用辭庫內。利用漢音工具箱的辭庫維護程式，可以皆理維護或更換不同的自用辭庫。

自用辭庫包含了3個主要的區域：[自勒學習](auto_learn_phrases.md)區、[自建辭](user_added_phrases.md)區、略語辭區。

自學習區保存了使用者用的同音字辭優先順序。自建辭區保存使用者自行登錄的自建辭。略語辭區保存略辭 及可展開之長短辭。

每一個略辭區含256個略語，每一個略語可以設定2個辭句。使用者只要翰入略語辭鍵，用 Tab 功能艇即可展開一個長辭或一個短辭。

本版的自用辭庫以案標籤來表示，而自建辭及略語均包合在同一個自用辭庫內。使用者只要指定不同的案標飯名稱即可選擇不同案的自建辭及自建略語。漢音輸入法提供一個預設的搞素標籤 “系統預設"。使用者亦得以建立任意名字禽檔案標籤的自用辭庫。

進入自用辭庫的方法如下：

1. 進入漢音工具箱。
2. 打開自用辭庫圖示。

關於自用辭庫的管理與維護如下：

## 辭庫標籤名稱

- 顯示目前所使用的辭庫名稱。

## 使用中辭庫的內容

- 顯示目前所示自用辭庫的自建辭字數與略辭數，以及剩餘可登錄的自建辭字數與略辭數
- 以滑鼠在欲選取的辭庫一下即可選取越辭庫

## 設定爲使用中的辭庫

- 以滑鼠選取辭庫後，按【設定為使用中的辭庫】接鈕
- 以滑鼠在敬選取的辭庫連續接兩下即可設定幫辭庫爲使用中的辭庫。
- 您可以建立好幾個自用辭庫，但每次只能使用一個。

## 增加新辭庫

- 以滑鼠選取辭庫後，按【增加新辭庫】按鈕。

## 更換辭庫名稱

- 以滑鼠選取辭庫後，鍵入辭庫之新名稱，再按【更換辭庫名稱】按紐。

## 刪除辭庫

- 以滑鼠選取辭庫後，按【刪除辭庫】按紐。使用中的辭庫不可刪除。
