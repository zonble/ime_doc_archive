# 特殊符號選擇表

提供使用者一些待的符號或圖案 ，在「特殊符號選擇」表內一共提供有439個特殊符號供使用者選擇。

利用快速功能鈕內的「特殊符號選擇」紐（或接Ctrl+3）。

例：請試著輸入「【電話】：755-6183」。

1. 將滑总指標移到快速功能鈕內的「特殊符號選擇」紐上，點接滑鼠左鍵一下（或接C1rl+3）。
2. 切換「特辣符號選擇」視窗，直到「【」符號出現。（切換方法豬參考選擇同音字（辭】）
3. 選擇「【」。
4. 輸入「電話」讀音及翰入「】」按【C1rl+】艇兩次切換到全形輸入模式（如果您是用拼音可省掉此步驟）。按〔Shift+：〕鍵輸入全形符號「：」。｛關於全形符號的輸入請參考輸入全形符號）
5. 按〔Enter〕 送出字串。
6. 確定Num Lock燈亮著，然後用九宮數字鍵輸入半形數字755-6183。

----

注意：當然您也可以參考附錄的「特获符號讀音對照表」，辣入越符號所對應的讀音，然後以選擇同音字的方式選取該符號。
