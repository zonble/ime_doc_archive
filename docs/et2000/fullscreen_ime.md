# 輸入法設定

在倚天全螢幕環境下，進入輸入法設定，可選擇：

1. 選擇輸入法<br>
   提供設定忘形、注音、倉頡、羅馬、三角、電信、詞庫、泰山、行列輸入法。<br>
   在每選定某種輸入法時，在輸入法參數上已有內定的參數設定。欲修改設定請利用滑鼠直接選擇。
2. 輸入法參數 提供指定輸入參考檔及造字參考檔的路徑、
    - [字根組錯嗶聲](fullscreen_input_error.md)、
    - [中英辨識](fullscreen_detect.md)、
    - [簡碼](fullscreen_code.md)、
    - [注音鍵盤](fullscreen_layout.md)、
    - [使用數字鍵盤選擇候選字功能](fullscreen_number.md)。
3. 狀態列：提供選擇使用[固定式、移動式](fullscreen_status_bar.md)方式。
4. 選擇候選字： 提供選擇[使用數字、字母](fullscreen_candidadte_keys.md)方式。
