# 略語展開功能

1. 將輸入視窗的內容清除（可按〔Enter〕）。
2. 輸入略辭鍵（二或三個中文字）。<br>
   若為正確的已登錄的略語辭鍵，則該字串下會顯示藍色波紋底線．
3. 按〔Tab〕可展開長辭、短辭。
4. 選擇要輸入長辭或短辭。
5. 按〔Enter〕鍵送出字串。

如果該中文字串未登錄在略辭區的話，則系統會發出嗶嗶的聲音，表示該字串不能做略語展開。當視窗顯示略語展開辭時，游標也暫時消失，這時按〔Enter〕鍵可送出該展開辭。

例：取用自建略「松下」的展開辭。

1. 輸入「松下」讀音。
2. 按〔Tab〕顯示其展開長辭及短辭。

漢音工具箱的略辭維護工具，提供對略語的新增、尋找、刪除、修改、瀏覽等功能。 請參考略辭維護。
