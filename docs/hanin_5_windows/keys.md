# 漢音功能鍵一覽表

漢音會預先設定快速鍵的組合，但您也可以修正這些快速鍵的定義。下列快速鍵自虛線後均可由使用者自行定義。

| 按             | 目的                                                                                               |
| -------------- | -------------------------------------------------------------------------------------------------- |
| Ctrl+Shift：   | 進入漢音輸入法，開啟漢音輸入視窗（或按 Alt+Shift)。                                                |
| Ctrl+Space：   | 關閉漢音輸入視窗。                                                                                 |
| Shift:         | 英數半形／漢音切換                                                                                 |
| →：            | 將漢音輸入視窗內的游標向右移動１個文字。                                                           |
| ←：            | 將漢音輸入視窗內的游標向左移動１個文字。                                                           |
| Home：         | 將游標移動到漢音輸入視窗內的字串首端。                                                             |
| End：          | 將游標移動到漢音輸入視窗內的字串尾端。                                                             |
| BackSpace:     | 刪除游標前之字或讀音。                                                                             |
| Enter:         | 送字串至應用軟體                                                                                   |
| Delete：       | 刪除一字。                                                                                         |
| Esc：          | 取消最後一次辭彙自動轉換；取消略語辭輸入狀態。                                                     |
| ↑：            | ‧取消漢音輸入視窗內中游標所在文字的最後一個讀音（即四聲）；<br>‧當游標位於漢音輸入視窗的字串尾端時，取消字串尾端最後一字的讀音（即四聲）。<br>‧執行同音字辭選擇時，顯示上一列的同音字辭。                                                        |
| ↓：            | ‧ 顯示輸入視窗內游標所在字辭的同音字辭；<br>‧ 當游標位於漢音輸入視窗的字串尾端時，顯示最後字辭的同音字辭 <br> ‧ 執行同音字辭選擇時，顯示下一列的同音字辭。                                                        |
| Space：        | ‧ 顯示漢音輸入視窗內游標所在字辭的同音字辭；<br>‧ 當游標位於漢音輸入視窗的字串尾端時，顯示最後字辭的同音字辭。|
| PgUp/PgDn:     | ‧換頁選擇同音字﹑辭。                                                                              |
| \：            | “\”即特殊符號功能鍵，表示進入鍵盤符號輸入。                                                        |
| Tab：          | 執行略語展開功能。                                                                                 |
| Shift+Enter：  | 執行“線上登錄自建辭”。                                                                             |
| Shift+Delete： | 清除漢音輸入視窗。                                                                                 |
| Ctrl+`：       | 切換輸入模式。可切換漢音輸入模式﹑全形英數輸入模式﹑半形英數輸入模式﹑地址輸入模式等四種輸入模式。 |
| Ctrl+?：       | 呼出曖昧音字集。                                                                                   |
| Ctrl+1：       | 展開／收縮快速功能鈕。                                                                             |
| Ctrl+2：       | 呼出文字查詢。                                                                                     |
| Ctrl+3：       | 顯示特殊符號表。                                                                                   |
| Ctrl+4：       | 呼出重複字庫。                                                                                     |
| Ctrl+5：       | 呼出『螢幕觸式鍵盤』。                                                                             |
| Ctrl+6：       | 呼出漢音工具箱。                                                                                   |
| Ctrl+7：       | 呼出線上輔助說明。                                                                                 |
| Ctrl+8：       | 設定快速功能鈕橫／直排。                                                                           |
