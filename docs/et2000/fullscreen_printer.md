# 印表機設定

進入印表機設定，可選擇：

1. 印表機清單 顯示區中有二種排列方式
    1. 依廠牌第一個英文字母排列。
    2. 依已知的印表機機型代號(Print Type)」排列。<br>
       共支援 180 餘種印表機，用滑鼠移動上下捲軸，選擇適合印表機驅動程式。
2. 螢幕列印

    - 「☉ 黑白 ○ 灰階 ○ 彩色」提供選擇螢幕拷貝色彩模式。
    - 「放大倍數」設定螢幕拷貝放大倍數模式。數值範圍 1∼8 倍。

3. 一般列印

    - 「列印埠」設定 LPT 輸出埠。數值範圍 1∼3。
    - 「錯誤重送」印表機產生錯誤時，利用此參數控制系統重送資料的次數。數值範圍 2∼255。
    - 「列印緩衝區」設定印表機緩衝區的大小。數值範圍 1∼30(Ｋ)。

4. 其他參數
    - 「列印預視」提供使用 OAmate 軟體具預視功能。
    - 「保護模式」將印表機程式移至 1MB 以上的記憶體。
    - 「使用上層記憶體」將 Upper Memory 打開，以增加主記憶體的運用。
    - 「由網路印表機列印」設定將資料從網路印表機印出。
