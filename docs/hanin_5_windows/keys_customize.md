# 自定快速鍵

原則上漢音會預先定義快速鍵，但使用者可配合自身需要，更改快速鍵的組合方式。更改快速鍵的組合必須根撼「案件定自方式說明」欄中的組合規則。在快速鍵的視窗中，自定快速鍵的方法如下：

1. 由控制台的翰入法管理員中選擇漢音輸入法的 5.0 版，接［內容］接鈕開啓漢音工具箱視窗。或經由快速功能鈕的「漢音工具箱」接鈕進入。出現「漢音輸入法」對話 。
2. 點按【快速鍵】按鈕。<br>若要返回工具箱只要再點按【工具箱】接鈕。
3. 從「項目」列示框中選擇飲執行更改的快速鍵。
4. 將滑指標移到「可選擇之快速艇定義」的對話框中，點接 1 下後直接翰入第一個代表字母鍵到翰入框中（只能 入［C］、［S］、［F］、［T］、［D］、［1］、［H］、［EJ、［PJ 等字母鏈）。
   - 此時「可選擇之快速 定義」翰入框中會根 步驟 4 的選擇，頭示以 Ctrl、Shifi、F 功能鍵、Tab、Delete、Insert、Home、End、PageUp、PageDown 等為開頭的快速鍵。
   - 「項目」列示框中有些快速艇定義無法更改，請參閱[漢音功能糙一覽表](keys.md)。
5. 由「可選擇之快速艇定義，下拉式選項表中選擇想更改的快速 組合。
6. 按【確定】執行鈕

［Shift］：英款半形/漠音切換、［→、←〕：翰入游標左右移動、［Home、End］：翰入游標移動至首尾、［BackSpace］：刪除游標前之字或讀音、［Enter］：送字串至應用軟艦、［Delete］：刪除一字、［Esc］：
取消愛換及關閉視窗、［十］：恢復讀音、［、Space］：選擇同音異義字、詞、［PgUp/Pgbn］：換頁選擇同音興義字，詞、［\］：進入〝糙盤符號”等快速艇無法變更。

---

注意：快速鍵無法重複定義。
