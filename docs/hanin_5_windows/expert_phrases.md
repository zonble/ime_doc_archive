# 專業辭庫

漢音系統辭典所蒐集的是一般常用的辭，對於特殊領域的辭彙則以專業辭典的方式提供給使用者，以適合各領域的使用者。漢音輸入法所附上的專業辭典有資訊、商業、運動、軍事、西醫﹑土木建築等，使用者可以線上環境設定或是以環境維護程式設定。對於需要地址及姓名輸入的使用者，漢音輸入法亦提供了地址(請參考地址輸入模式）及姓名辭典（請參考姓名辭典），蒐集了本省縣市鄉鎮名、常用的街道以及對於姓名常用字優先轉換，以提高地址姓名輸入的正確轉換率，在含有地址姓名資料的輸入過程中，得以隨時和系統辭典配合使用。

關於專業辭庫的詳細說明如下：
漢音除了地址辭典及姓名辭典外，另外又提供了１４種專業辭典供使用者選擇。不過使用者每次最多只能選用３種。

---

\## 注意：選用專業辭典，應該配備更高速的電腦、更多的記憶體，以增進執行效率。

---

■ 選用方法

1. 打開工具箱。
2. 將滑鼠指標移到標示有「醫」字圖示的辭典。<br>
   此時會出現「專業辭典選擇」的視窗。
3. 將滑鼠指標移到要選用的辭典上後點按滑鼠左鍵一下。
4. 再選擇【選用】鈕。<br>
   此時被選到的辭典便會顯示在右邊的「找詞的次序」的顯示框內。

■ 設定找詞的優先順序

漢音找詞的順序為使用者選用的專業辭典優於系統辭典，最後為姓名辭典。由於使用者一次可選用３種辭典，而找詞的優先順序也會因為辭典的排列不同而有所更動。若您要更改專業辭典的先後順序，其步驟如下：.

1. 將滑鼠指標移到要上移（或下移）的辭典上，按下滑鼠左鍵，使其反白。
2. 選擇【往上移】（或【往下移】）。

■ 清除

清除：清除單本辭典。

全部清除：清除所有選到的辭典。
