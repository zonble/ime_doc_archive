## 選擇字形大小與顏色

漢音輸入視窗內的顯示文字，可依使用的客好重新調整字體．大小及顏色．

例：字形屬性設定

1. 開啓漢音工具箱-
2. 在電腦的畫像上按一下。<br>出現「漢音屬性設定」對話框。
3. 點按左上方的【字形】接鈕。<br>出現字形設定畫面
4. 選擇所要的字形。大小與顏色後，再接下【確定】鈕即可

設定字形請選擇已安装的中文字形或 System 字形，否則字形可能無法顯示出來。顏色設定可系分為文字顏色。
同音字顏色與讀音顏色等三種 。字形大小的設定销固是 8~26。 有些字形可能只支援某些销固的字，超出範圖的話，漢音可能無法顯示
