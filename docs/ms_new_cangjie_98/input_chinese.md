## 輸入中文

在大部分的情況下，使用倉頡輸入只需直接輸入字碼。只有在少數情況下，當該組字碼含有
一個以上的字或當您使用 \* (萬用字元)代替忘記的字碼時，才有可能需要選字。此時，新
倉頡輸入法會在您需要選字的時候，自動為您選擇正確或常用的字詞。如果在完成輸入之前
(您可以使用 [字尾符號](endsymbol.md) 或 Enter 鍵來完成輸入)發現有任何字詞需要修
改，您可以移動游標至該字並使用方向鍵來進行編輯工作。有關如何修改輸入的文字，請參
閱[挑選同碼字詞](pickup.md) 。本輸入法可以自動調整的前後最長字元為三十一個字，超
過三十一個字後，輸入法會自動將句子前面的長詞送出。

另外，如果您在輸入的過程中，忘記某個字的字碼拼法，您可以使用 \*
(萬用字元) 來輔助您。只要將您忘記的字碼以 \* 代替即可。下列為所有使用
'\*' 輸入倉頡碼的可能性：

A\* ; A\*A ; AA\* ; AA\*A ; AAA\* ; AAA\*A ; AAAA\*

A 代表一個倉頡碼，\* 可代表 0~4 個倉頡碼。

**提示**

\* 不能置於字碼的第一位，並且在其後最多只能有一個字碼。
