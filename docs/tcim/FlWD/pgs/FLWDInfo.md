# 輸入法介面介紹

若現用語系為中文，則鍵盤的輸入會先透過輸入法介面的處理，並根據所使用的中文輸入法的組碼原則，組合成中文字或其他字元。

中文系統選取了在形、音、義三方面較具代表性的輸入方法，包括：[倉頡](../../CJJY/pgs/CJJYInfo.md)、[簡易](../../CJJY/pgs/CJJYJYIF.md)、[大易(詞庫版)](../../CJJY/pgs/DaYiInfo.md)、[拼音](../../CJJY/pgs/PYJPInfo.md)和[注音](../../CJJY/pgs/ZhuYInfo.md)等。

中文系統的輸入法，使用共同的介面設計和鍵盤控制，包括：[輸入窗](FLWDIW.md)、[選字窗](FLWDDW.md)和[操控板](../../Menu/pgs/MenuPale.md)。
