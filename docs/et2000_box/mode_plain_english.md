## 純英文模式

中文模式與英文模式互換鍵

《使用方式》

請選一種操作方式

-   -▲ 選「設定」下拉表直接選純英文模式，在前方會出現「ˇ」。
-   ▲ 用滑鼠選 ![bm6.BMP](./bm6.png) 按鈕。
-   ▲ 用鍵盤按[Alt]-[Ctrl]-[A]鍵。

開機並且進入倚天中文視窗模式後，即處於『中文模式』；如果您想查看中文字的 ASCII 碼，或改變系統的執行模式，可以切換到『英文模式』下，最明顯的改變就是原本在畫面上的中文字全部以 ASCII 碼顯示。

但切換到英文模式時，主記憶體仍存有倚天中文視窗，只是不能發揮正常的中文處理功能了。

簡單地說，在英文模式下，可以輸入中文、英文、數字，但全部以英文顯示；而切換到中文模式下，則可使用系統提供的各項功能。
