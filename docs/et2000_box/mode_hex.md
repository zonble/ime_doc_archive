# 內碼輸出

十六進位模式互換鍵(Hexadecimal Mode Toggle)

《使用方式》

請選一種操作方式

-   ▲ 選「設定」下拉表直接選內碼輸出，在前方會出現「ˇ」。
-   ▲ 用滑鼠選 ![bm3.BMP](./bm3.png) 按鈕。
-   ▲ 用鍵盤按[Alt]-[Ctrl]-[H]鍵。

本功能是將您所輸入的中文字的內碼顯示在螢幕上。如此一來，對於您想查詢某個中文字的內碼會顯得非常的方便。

在內碼為 BIG-5 碼時，想查詢『倚』字的內碼，切換內碼輸入功能，使用倉頡輸入法，輸入「倚」字的倉頡碼「人大一口」，此時螢幕會出現倚字的內碼「 ADCA 」。

在不同的內碼系統下，即使是同一個字，出現在螢幕上的內碼也不同；如在公會碼系統下，「倚」字的內碼是「 9AAC 」。
