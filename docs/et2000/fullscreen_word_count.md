# 字數設定

在倚天全螢幕環境下，進入字數設定，可選擇點陣字形環境設定輸入使用數值。

1. 16×15 字形載入字數<br>系統預設值為 5401 字，如果要改變，請於資料輸入區輸入數值，數值範圍由：1∼(字)。
2. 16×15 字形緩衝區字數<br>系統預設值為 16 字，如果要改變，請於資料輸入區輸入數值，數值範圍由：1∼(字)。
3. 16×15 字形造字載入字數<br>系統預設值為０字，如果要改變，請於資料輸入區輸入數值由 1∼8191。
4. 24×24 字形載入字數<br>系統預設值為０字，如果要改變，請於資料輸入區輸入數值，數值範圍由：1∼(字)。
5. 24×24 字形緩衝區字數<br>系統預設值為 16 字，如果要改變，請於資料輸入區輸入數值，數值範圍由：1∼(字)。
6. 24×24 字形造字載入字數<br>系統預設值為０字，如果要改變，請於資料輸入區輸入數值由 1∼8191。

當您設定所載入的字數越多，系統處理的速度越快，但相對的所佔的記憶體就越大。
