# 倚天全螢幕

倚天全螢幕提供下列設定功能：

1. [輸入法設定](fullscreen_ime.md) 提供選擇使用輸入法，配合的輸入法參數設定、狀態列設定及選擇候選字方式。
2. [印表機設定](fullscreen_printer.md) 提供選擇印表機清單，螢幕列印設定、一般列印設定及其他參數設定。
3. [字數設定](fullscreen_word_count.md) 提供配合硬體及記憶體需求，彈性運用點陣字形載入字數，可調整中文系統佔用記憶體的情形。
4. [其他參數設定](fullscreen_other.md) 提供中文相關功能設定、使用內碼設定及將部份程式移入顯示記憶體的選項。
