# 自動學習

漢音對同音字、辭有自動學習能力，減少再次選擇的機會。

漢音在您每次選擇同音字（辭）時，會自動記住您選擇的字（辭）作為下次相同讀音再出現時的最優先選擇、原來的最優先選擇鍵成次優先，會置於同音子（解）選擇列的最前方，所以，您越近用過的，您越容易得到。

例：輸入下列兩行字串：

```text
「程式名稱：張兆樺」
「程式貌明：執行此程式可顯示張兆樺個人資料」。
```

1. 連續輸入第一行讀音。
2. 選同音字，「划」改為「華」。「照」改為「兆」．選「城市」同音辭「程式」，按【Enter〕鍵送出字串。
3. 連續輸入第二行讀音，按〔Enter〕鍵送出字串。

上例中，您在連續輸完第一行字串的讀音後，必須做二次同音字及一次同音辭選擇才得到正確字串，輸入第二行字串時，雖然同樣的字與辭再度出現，但您卻完全不用再做選擇，這就是字辭優先的自動學習。
