# 曖昧字音集

對於一些平常大家容易混淆的讀音，常會造成中文注音輸入的困難，因此漢音輸入法提供了曖昧處理。使用者按下曖昧處理的功能鍵，系統將替使用者找出最接近的讀音，同時進入同音字辭的選擇模式。

『曖昧變換』是將指定的中文字音列出其相似的發音組合，並由使用者選擇取代之，而這些近似的發音組合稱為曖昧字音集。ㄓ與ㄗ、ㄔ與ㄘ、ㄕ與ㄙ、ㄣ與ㄥ…等都是常令人混淆易生錯誤的近似音，一旦輸入錯誤，則可經由曖昧變換進行修正，免除重新輸入的麻煩。例如「ㄘ ㄈㄢ ˋ」（吃飯）。

執行『曖昧變換』的功能鍵為〔Ctrl+?〕。

〔Ctrl+?〕：選取游標所在字的下一個曖昧字集，並列出此字集內之所有同音字或辭。繼續按此鍵則重複上述之動作，直到最後一個曖昧字集再從頭開始。

例：使用曖昧變換

1. 輸入「ㄘ ㄈㄢ ˋ」。<br>
   假設輸入視窗出現『雌飯』。
2. 移動游標在「雌」字位置，按下〔Ctrl+?〕，再選擇１，則以『吃飯』置換掉原辭。
