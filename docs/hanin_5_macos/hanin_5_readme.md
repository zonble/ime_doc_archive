## 新增功能：

1. 系統辭典增加為 6 萬 了千個詞。
2. 增加 14 類（如商業、資訊、法律政治 ⋯ 等）專業辭典，共 120,000 詞。

## 安裝說明：

1. 漢音輸入法 v5.0 共包含三個部份： -（1） 主程式：Hanin v5.0 -（2） 設定面板：HaninSetup v5.0 -（3） 辭典：3 個主要辭典以及 14 個專業辭典
2. 將漢音辭魚輸入法主程式 “Hanin V5.O”拷貝到“系統檔案夾”（System Folder）內的“延伸功能”
   （Extensions）檔案夾內。
3. 將漢音設定控制面板 “HaninSetup v5.O”拷貝到“系統檔案夾”（System Folder） 內的“控制面版”（Control Panels） 檔案夾内。
4. 將漢音系統辭典“HaninDic”，地址辭典 “AdrDic”，曖昧音辭典 “AimeiDic”拷貝到“系統檔案夾”（System Folder） 裡面的“Hanin5”檔案夾內。
5. 將 14 個專業辭典拷貝到“系統檔案夾”（System Folder）裡面的“Hanin5”檔案夾裡面的“Expert” 檔案夾內。
6. 重新開機，即可開始使用漢音。

## 功能使用說明：

1. 地址模式與一般模式切換：<br>
   當您要輸入地址或人名時，由漢音的選單（Pen Menu）選擇地址模式，或是按 option+shift+A，切換至地址模式後再輸入，將可提高辨識率，若要恢復一般模式時，由漢音的選單（Pen Menu）選擇一般模式，或是按 option+shift+Z。
2. 選擇倚天注音為輸入方式：<br>
   由漢音的選單（Pen Menu）選擇漢音設定，或是打開“控制面版”檔案夾內的漢音設定檔案，然後用滑鼠按一下倚天的圖像即可。
3. 暧味音變換功能：<br>
   曖味音變換是指將每一個中文字的發音，取其最類似的發音組合，譲使用者選擇・例如：“イー”的類似音有“イメジコ“ちパ・“ちメー等・
   當您要選擇曖昧音時，將游標移到要變換的中文字後面，然後按 shift+space bar，就會出現選宇視窗讓您選字，或是按空白鍵進入同音字視窗後，再用滑鼠按一下左上角曖昧音按鈕。一且進入了選字視窗後，您可以按 Tab 鍵，或是用滑鼠按一下中文字的讀音來更換不同的曖昧音。
4. 自建詞登錄：<br>
   將游標移到浮動視窗中文字串倒數的 2-5 字前，按 shift+return 即可登錄。
5. delete 鍵：<br>
   按 delete 鍵時會把游標前之中文字的讀音還原，若要刪除前一個中文字，則要按 shift+delete 鍵。
   X 如要切換 delete 鏈與 shift+delete 鍵的使用功能，可在漢音設定控制面板中，按一下漢音圖像，在漢音辭輸入法版權說明下方，選擇「刪除鏈不還原讀音」，則按 delete 鍵時會把游標前之中文字刪除，按 shift+delete 鏈可把游標前之中文字的讀音還原。

× 注意事項：

一旦您覺得漢音的變換不正常時，請檢查漢音輸入法的選單，是否設定為地址模式，如果您不是正在輸入地址或人名時，把它切換為一般模式就會正常了。
